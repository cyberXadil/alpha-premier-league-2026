⚙ One-Time Firebase Setup — Takes About 3 Minutes
Step 1 · Go to console.firebase.google.com → Click "Add Project" → Name it anything (e.g. alpha-premier-league).
Step 2 · Inside the project, click the Web icon ( </> ) → Register your app → Copy the firebaseConfig values shown.
Step 3 · Go to Build → Firestore Database → Click "Create Database" → Choose "Start in test mode".
Step 4 · Go to Build → Storage → Click "Get Started" → Choose "Start in test mode".
Step 5 · Open this HTML file in any text editor (Notepad / VS Code), find the section labelled YOUR FIREBASE CONFIG, and paste your values.

The section to edit looks exactly like this:
// ── PASTE YOUR FIREBASE CONFIG HERE ── const firebaseConfig = { apiKey: "YOUR_API_KEY", authDomain: "YOUR_PROJECT_ID.firebaseapp.com", projectId: "YOUR_PROJECT_ID", storageBucket: "YOUR_PROJECT_ID.appspot.com", messagingSenderId: "YOUR_SENDER_ID", appId: "YOUR_APP_ID" };
